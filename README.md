# AtcoderCpp

## Overview: 競技プログラミングAtCoderの練習リポジトリ
- Atcoderの練習のために残しているものです。
- Problemディレクトリ内にこれまで解いてきた問題があります。
- github上のipynbファイルから直接notebook内のプログラムを伺えます。
- 言語は全てc++で書かれてあります。

## Architecture:
- AtcoderCpp
  - Dockerfile
  - docker-compose.yml
  - AtcoderProgram
    - Utility
      - STL.ipynb
      - DynamicProgramming.ipynb
      - bit.ipynb
    - Problems
      - Atcoder初級編
